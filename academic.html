  • Firebase Authentication

Coin App

Coin App is a simple iOS application built with SwiftUI that displays a list of popular cryptocurrencies using the [CoinPaprika API](https://api.coinpaprika.com). Users can tap on any coin to view detailed information including its logo, name, symbol, start date, and hash algorithm.

Technologies Used
  • SwiftUI
  • MVVM Architecture
  • CoinPaprika API
  • Alamofire (for HTTP requests)

BookStore App

The BookStore App is a native Android application that allows users to manage a personal library. The app features a complete CRUD system (Create, Read, Update, Delete) using a local database. It was developed as part of a learning journey in native Android development with Kotlin and Jetpack components.

Technologies Used
  • Kotlin
  • Room Database – Local data persistence with SQLite support
  • Android SDK
  • ViewModel + LiveData – MVVM pattern implementation
  • XML Layouts – UI structure
  • Android Studio – IDE
